# CodeAlpha_EDA
Overview

This project focuses on performing Exploratory Data Analysis (EDA) using SQL. The aim is to analyze structured data directly within a database to uncover patterns, trends, and key insights without relying on external tools.

Tools & Technologies
MS SQL (Structured Query Language)

Features
Filtering and grouping data for better understanding
Aggregations such as counts, averages, and sums
Identifying trends, patterns, and anomalies
Creating reusable queries for analysis
Key Analysis Performed

Purpose
This project demonstrates how SQL can be used not just for data storage, but also for in-depth data analysis. It highlights the ability to extract insights efficiently using structured queries.

Output
The project produces query-based results that summarize data, highlight trends, and support better decision-making.
